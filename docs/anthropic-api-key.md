## Anthropic API-Key holen

1. Öffne [Claude Console](https://platform.claude.com/).
2. Melde dich an oder erstelle ein Konto.
3. Gehe zu **API Keys** und klicke auf **Create Key**.
4. Gib dem Key einen Namen.
5. Kopiere den Key sofort und füge ihn in Plotter Studio ein.

Tipp: Der Key wird nur einmal vollständig angezeigt und sollte nie öffentlich geteilt werden.
